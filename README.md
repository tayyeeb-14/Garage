# SpeedX Garage

A production-ready foundation for a garage management system built with React Native, React, Node.js, Express, TypeScript, MongoDB Atlas, and JWT.

## Structure

- backend: REST API, MVC architecture, authentication-ready services, middleware, and utilities
- admin: React + TypeScript admin panel scaffold
- mobile: React Native + Expo customer app scaffold

## Getting Started

1. Copy backend/.env.example to backend/.env and fill in environment values.
2. Install dependencies with npm install.
3. Start the backend with npm run backend:dev.

## Notes

This scaffold intentionally focuses on the complete project foundation only and does not introduce business features yet.
